## ⏮ 开篇：先把画面装进脑子

> 一张清晰照片，你一层层撒"雪花噪点"，撒上千步，最后变成纯雪花屏。
> 这个**加噪**过程固定、不用学。反过来——若有网络能"看着带噪图、猜出刚撒上去那层雪花"，就能减掉它、让图变清楚一点；从纯噪声减上千次，就**生成**出一张图。
> 网络要学的只有一件事：**给一张带噪图，猜出那层噪声**。

下面 8 步，就是把这句话**翻译成精确的数学**。每个公式出现时，你只要问一句"它在说上面这段话的哪一句？"

记号约定：`x_0` 是干净图，`x_t` 是第 t 步的带噪图，`t = 1…T`（T 通常 1000），`ε` 表示标准正态噪声 `N(0, I)`。

---

## 🎯 主菜

### 台阶 1：前向一步加噪 —— 写清楚"撒一层雪花"

定义"从 `x_{t-1}` 到 `x_t`"这一步：

```
q(x_t | x_{t-1}) = N( x_t ; √(1-β_t)·x_{t-1} ,  β_t·I )
```

逐符号解释：
- `β_t`：第 t 步**加多少噪声**，一个 0~1 之间的小数（noise schedule，人为设定，如从 0.0001 线性涨到 0.02）。
- 均值是 `√(1-β_t)·x_{t-1}`：把上一步的图**按 √(1-β_t) 缩小一点**（信号衰减）。
- 方差是 `β_t`：再**掺入 β_t 这么多新噪声**。

写成采样式（这就是 reparameterization：高斯采样 = 均值 + 标准差×标准正态）：

```
x_t = √(1-β_t)·x_{t-1} + √(β_t)·ε ,     ε ~ N(0, I)
```

**为什么均值要乘 √(1-β_t)、而不是直接加噪？** 为了让总能量稳定：若 `x_{t-1}` 方差是 1，则 `x_t` 方差 = `(1-β_t)·1 + β_t = 1`。信号一点点让位给噪声，但**总方差始终≈1**（这叫 variance-preserving），数值稳定。

---

### 台阶 2：前向"一步到位" —— 不想撒 1000 次，能不能直接从 x_0 跳到 x_t？

能。令 `α_t = 1-β_t`，台阶 1 变成 `x_t = √(α_t)·x_{t-1} + √(1-α_t)·ε`。把它**往前代一层**（把 `x_{t-1}` 也展开）：

```
x_t = √(α_t)·[ √(α_{t-1})·x_{t-2} + √(1-α_{t-1})·ε' ] + √(1-α_t)·ε
    = √(α_t·α_{t-1})·x_{t-2}  +  [ √(α_t(1-α_{t-1}))·ε' + √(1-α_t)·ε ]
```

看方括号那两项:**两个独立的零均值高斯噪声相加,结果还是一个高斯,方差直接相加**:

```
方差 = α_t(1-α_{t-1}) + (1-α_t) = 1 - α_t·α_{t-1}
⟹  方括号 = √(1 - α_t·α_{t-1})·ε̄ ,   ε̄ ~ N(0, I)
```

于是 `x_t = √(α_t α_{t-1})·x_{t-2} + √(1-α_t α_{t-1})·ε̄`。形式和上一行**一模一样**，只是少了一层。一路代到 `x_0`，记 `ᾱ_t = α_1·α_2·…·α_t`（连乘），得到**全篇最重要的一条闭式**：

```
x_t = √(ᾱ_t)·x_0 + √(1-ᾱ_t)·ε                    ……(★)
```

意义：① 任意 t 一步采样，训练时不用迭代；② t 越大 `ᾱ_t→0`，`x_t` 越接近纯噪声 `ε`——和"撒到最后变雪花屏"对上了。

---

### 台阶 3：反向 —— 我们真正想要的，和它的麻烦

生成 = 反着走，需要"去噪一步"的分布 `q(x_{t-1}|x_t)`。但它**算不出来**（要遍历整个数据分布才知道）。所以用一个神经网络去近似它：

```
p_θ(x_{t-1}|x_t) = N( x_{t-1} ; μ_θ(x_t,t) , Σ_θ )
```

`θ` 是网络参数，`μ_θ` 是它预测的均值。**怎么训练 θ？** 自然想法：最大化训练图的似然 `log p_θ(x_0)`。但它也算不出来（要把所有中间 `x_1…x_T` 积分掉）。两条路都堵死——台阶 4 用一个标准武器绕过去。

---

### 台阶 4：ELBO —— 把"算不出的似然"换成"能优化的下界"

当似然算不出，就优化它的**变分下界 ELBO**（证据下界）。结论形式（这是 DDPM 论文经过整理的版本）：

```
-log p_θ(x_0) ≤ L_T + Σ_{t=2}^{T} L_{t-1} + L_0

其中  L_{t-1} = KL( q(x_{t-1}|x_t, x_0)  ‖  p_θ(x_{t-1}|x_t) )
```

逐项读：
- `L_T = KL(q(x_T|x_0) ‖ p(x_T))`：终点要像标准高斯。**无参数、几乎为 0**，忽略。
- `L_0 = -log p_θ(x_0|x_1)`：最后一步的重建项，细节项。
- `L_{t-1}`（**核心**）：让网络的去噪分布 `p_θ`，去逼近 `q(x_{t-1}|x_t, x_0)`。

**关键转折**：注意核心项里是 `q(x_{t-1}|x_t, x_0)`——**多了一个 `x_0` 条件**。单凭 `x_t` 那个后验算不出来，但**额外告诉它干净图 `x_0`**，它就**变得可解析**了。台阶 5 就来算它。

---

### 台阶 5：那个"带 x_0 的后验"，其实是个高斯（可解析）

用贝叶斯公式拆开（中间用了前向的马尔可夫性 `q(x_t|x_{t-1},x_0)=q(x_t|x_{t-1})`）：

```
q(x_{t-1}|x_t,x_0) = q(x_t|x_{t-1}) · q(x_{t-1}|x_0) / q(x_t|x_0)
```

右边三个**全是已知高斯**：`q(x_t|x_{t-1})` 来自台阶 1，另两个来自闭式 (★)。三个高斯的概率密度相乘，指数部分是 `x_{t-1}` 的二次函数——**把它按 `x_{t-1}` 配成完全平方**（标准操作：二次项系数定方差，一次项定均值），就得到一个高斯 `N(μ̃_t, β̃_t I)`：

```
β̃_t = (1-ᾱ_{t-1})/(1-ᾱ_t) · β_t                                  (方差)

μ̃_t = [√(α_t)(1-ᾱ_{t-1})/(1-ᾱ_t)]·x_t  +  [√(ᾱ_{t-1})·β_t/(1-ᾱ_t)]·x_0   (均值)
```

（逐行配平方的代数有点占地方，想看我可以单独展开；这里你只需接受"三个高斯相乘配方 ⟹ 一个高斯"，并记住均值 `μ̃_t` 同时依赖 `x_t` 和 `x_0`。）

---

### 台阶 6：把 x_0 换成噪声 ε —— 让均值只跟"噪声"有关

`μ̃_t` 里有 `x_0`，但生成时我们没有 `x_0`。用闭式 (★) 反解出 `x_0`：

```
由 (★):  x_t = √(ᾱ_t)·x_0 + √(1-ᾱ_t)·ε
反解:    x_0 = ( x_t − √(1-ᾱ_t)·ε ) / √(ᾱ_t)
```

代进台阶 5 的 `μ̃_t`，合并 `x_t` 的系数。**精华的一步**是这个化简（用了 `ᾱ_t = ᾱ_{t-1}·α_t`）：

```
x_t 的系数合并后分子 = α_t(1-ᾱ_{t-1}) + β_t = α_t − ᾱ_t + (1−α_t) = 1 − ᾱ_t
```

分子恰好把分母 `(1-ᾱ_t)` 约掉，于是一切坍缩成无比干净的：

```
μ̃_t = (1/√(α_t)) · ( x_t − (β_t/√(1-ᾱ_t))·ε )                  ……(♣)
```

读它：**只要知道当前带噪图 `x_t`、和"这一步混进去的噪声 `ε`"，就能算出后验均值。** —— 噪声 `ε` 成了唯一的未知数。

---

### 台阶 7：让网络去预测 ε —— KL 坍缩成一个 MSE

既然真均值 (♣) 长这样，就让网络的均值 `μ_θ` **照抄同一个公式，只把真噪声 `ε` 换成网络的预测 `ε_θ(x_t,t)`**：

```
μ_θ = (1/√(α_t)) · ( x_t − (β_t/√(1-ᾱ_t))·ε_θ(x_t,t) )
```

把方差设成一样，那么两个同方差高斯的 KL **就只剩均值差的平方**：`KL = ‖μ̃_t − μ_θ‖² / (2σ_t²)`。两个均值相减，`x_t` 完全抵消，只剩噪声差：

```
μ̃_t − μ_θ = (1/√(α_t))·(β_t/√(1-ᾱ_t))·( ε_θ − ε )

⟹  L_{t-1} = [ β_t² / (2σ_t²·α_t·(1-ᾱ_t)) ] · ‖ ε − ε_θ(x_t,t) ‖²
```

绕了一大圈，每个时间步的损失，**本质就是"真噪声 ε 和预测噪声 ε_θ 的均方误差"**，前面那一坨只是个权重系数。

---

### 台阶 8：丢掉权重 → L_simple，回到那句大白话

DDPM 论文发现：把台阶 7 那个权重系数**直接扔掉、设为 1**，训练反而更稳、生成更好。于是最终训练目标就是：

```
L_simple = E_{ x_0, t, ε } ‖ ε − ε_θ( √(ᾱ_t)·x_0 + √(1-ᾱ_t)·ε ,  t ) ‖²
                              └──────────── 就是 (★) 的 x_t ───────────┘
```

把它读成中文：**随便拿张图 `x_0`、随机挑一步 `t`、随机撒一层噪声 `ε` 造出带噪图 `x_t`，让网络看着 `x_t` 把 `ε` 猜出来，算 MSE。** —— 这就是开篇那句"给一张带噪图，猜出那层噪声"。**8 个台阶绕回了原点。**

> 所以那句被我上一版直接砸给你的"L_simple 是 reweighted ELBO"——现在你知道它的全名了：台阶 4 的 ELBO → 台阶 7 的"权重×MSE" → 台阶 8 扔掉权重。

### 实现：训练就两行核心（PyTorch 伪代码）

```python
def train_step(x0, model, abar):                 # abar[t] 预存 ᾱ_t
    t   = torch.randint(0, T, (x0.size(0),), device=x0.device)
    eps = torch.randn_like(x0)                    # 随机撒的噪声 ε（答案）
    at  = abar[t].view(-1, 1, 1, 1)
    xt  = at.sqrt()*x0 + (1-at).sqrt()*eps        # 闭式 (★) 造带噪图
    return F.mse_loss(model(xt, t), eps)          # 让网络猜 ε —— L_simple
```

## 🔗 与你 CVPR 工作的连接（一句话）

PolarAnything 这类 conditional DDPM，就是把 `ε_θ(x_t,t)` 加一个条件变成 `ε_θ(x_t,t,c)`（c=RGB）。其余 8 个台阶**一字不改**——这就是为什么你能拿 DDPM 的现成框架去做偏振生成。而反向每步"预测并减去噪声"，和你的 **confidence-guided 残差修正**在结构上是同一种"预测残差再修正"的思想。

## 🔧 配菜·面试（现在你能自己答了）

**Q：** DDPM 训练为什么是个预测噪声的 MSE？ELBO 那堆 KL 去哪了？
**A：** 顺着台阶 4→7→8 讲：ELBO 的核心是每步两个高斯的 KL（台阶 4），因为带 x_0 的后验是解析高斯（台阶 5）；把均值用噪声参数化后（台阶 6-7），KL 坍缩成 `权重·‖ε−ε_θ‖²`；DDPM 把权重设 1（台阶 8）得 L_simple。**它不是没有 ELBO，是 ELBO 的 reweighted 简化版。**

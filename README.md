# claudecode_learnword · 每日知识推送

每天推送服务于「**科研深耕 + 大厂算法岗**」的硬核知识，沿研究主线由浅入深、成体系积累。
不是碎片化的趣味知识，而是一年后能直接变现为面试硬实力与科研纵深的「知识复利」。

> 现在分**两条平行线**，各自独立推进、互不干扰：
> - 🎯 **技术线**（主线）：科研主线深挖 + 面试八股 —— 见 `ROADMAP.md` / `daily/`
> - 🌍 **人文线**（通识）：地缘 / 历史 / 经济 / 政治 / 文明 —— 看懂当下世界、攒谈资、换脑子，同样成体系 —— 见 `humanities/`

- 路线：可控密集预测 → 可控生成 → 世界模型
- 目标：多模态生成 / 世界模型方向算法岗，毕业进大厂

## 怎么用

### 手动（即时交互，推荐日常用）

打开本项目，按想看哪条线说一句话：

- 🎯 **技术线**：说 **"今天的"** / **"来今天的知识"**，或 `/daily-knowledge` → 生成 `daily/YYYY-MM-DD.md`
- 🌍 **人文线**：说 **"今天的人文"** / **"来点通识"**，或 `/daily-humanities` → 生成 `humanities/posts/YYYY-MM-DD.md`

两条线流程一致：读取各自进度 → 生成当天篇 → 回写对应的 `STATE.md` / `INDEX.md`。
生成后都可**当场追问、要展开某点、说"太浅/太深"调难度、说"换个方向/换模块"**。

### 自动（已配置 ✅，本地定时，无需 git / GitHub）

用本地定时任务实现，文件直接写入本项目：

| 任务 | 线 | 时间（本地） |
|---|---|---|
| `daily-tech-knowledge` | 🎯 技术线 | 每天约 10:00 |
| `daily-humanities-push` | 🌍 人文线 | 每天约 21:00 |

- 机制：在本地 Claude Code 内定时触发，每次自动读各自 `STATE.md` 接着上次讲，**不依赖 git/GitHub**。
- 要求：到点时 **app 开着**；若没开，**下次打开会自动补跑**。
- 管理：侧边栏 **Scheduled** 区可暂停 / 改时间 / 删除 / 立即运行；或直接跟我说"把人文线改隔天 / 暂停技术线"。
- 注意：首次自动运行时可能弹一次"允许读写项目文件"的权限框，点允许后会记住，以后不再打扰。

## 内容设计

### 🎯 技术线（每天）

| 板块 | 频率 | 说明 |
|---|---|---|
| 主菜·主线深挖 | 每天 | 路线图当前概念：直觉 → 数学 → 实现 → **与我 CVPR 工作的连接** |
| 配菜·面试考点 | 每天 | 一道 ML/DL/CV 八股或手撕，尽量呼应当天主菜 |
| 前沿速览 | 每周 2–3 次 | 一篇相关 arXiv 新论文 3 分钟速览（需联网） |

难度基线：面向研究生、已懂扩散基础，**不从零讲**。随时可调。

### 🌍 人文线（每周 2–3 篇）

| 板块 | 说明 |
|---|---|
| 开篇·画面/问题 | 用一个场景或反差把你拽进来 |
| 主线·来龙去脉 | 抓大转折、理因果链，一篇只咬一条主线（**不堆年表事实**） |
| 🌍 与当下的连接 | 这段知识怎么解释今天的某条新闻 |
| 💬 谈资点 / 洞见 | 一个能在饭桌或面试抛出的角度或金句 |

铁律：**脉络优先**；**政治话题中立**（给事实 + 多方视角，不灌输立场，养判断力）；比技术线**更口语、能换脑子**。
当前从 **H2 地缘政治** 起步，路线见 `humanities/ROADMAP.md`。
